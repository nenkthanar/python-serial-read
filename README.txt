This python application for Read and write serial data through USB to Serial or else

### First step
  1 Use arduino IDE for ubload arduino-serial.ino in folder name arduino-serial(use arduino mega or uno or any kind of arduino )
  
### Python libraries requirement ,pip command below
  1 pip install pyserial 

### How to run test this python application
  1 Connect USB to RS 232 or USB to Arduino board
  2 Go to root project by command promt on windows or terminal on linux , ubuntu 
    or Visual Studio Code Application terminal excamples => cd python-serial-read
  3 Use command  => python app.py
